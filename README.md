### [renRAKU](https://www.renrakuchou.com/): Communicate more, do less.

renRAKU replaces [連絡帳](https://renrakucho.net/eng/introduction/intro-3.html)(*Renrakuchou*), paper note books that teachers in Japan use to communicate with parents. It allows writing daily notes to parents/teachers, uploading of schedules, creation of school/class events, posting grades for individual students, and scheduling of parent/teacher meetings.

renRAKU uses the following technologies:

- Ruby on Rails
- CSS 3
- HTML 5
- JavaScript ES6
- PostgresSQL
- Heroku
- Git/Github
